**Description**

Export Tabs as Shortcuts turns the tabs in your current window into real,
double-clickable shortcut files — no popup, no setup, no account.

Click the toolbar button and every open tab is saved to your Downloads folder
as a shortcut file you can keep, organize, back up, or reopen any time:

• macOS → .webloc ("Web internet location") files
• Windows & Linux → .url internet-shortcut files

Double-click any saved file to reopen that page in your default browser.

WHY YOU'LL LIKE IT
• One click — exports the whole window at once, with a badge showing the count.
• Tidy — files land in a "Web Links" folder, named after each page's title.
• Incognito aware — tabs from private windows go into a separate "Private"
  subfolder.
• Private by design — works entirely on your device. No servers, no tracking,
  no network requests of any kind.
• Lightweight — internal pages (chrome://, about:, extension pages) are skipped
  automatically.

PERMISSIONS
• "tabs" — to read the title and URL of each open tab.
• "downloads" — to save the shortcut files to your computer.
That's it. The extension never sends your data anywhere.

To export from incognito windows, enable "Allow in incognito" on the
extension's Details page.

**Single purpose** (required justification)
The extension has one purpose: export the open tabs of the current window as
local shortcut files (.webloc / .url) that reopen each page in the browser.

**Permission justifications**
- tabs: Needed to read the title and URL of each open tab in the current
  window so they can be written into shortcut files.
- downloads: Needed to save the generated .webloc / .url shortcut files into
  the user's Downloads folder.
